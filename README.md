# yys-tool

## Project setup
```bash
yarn
# or
npm i
```

### Compiles and hot-reloads for development
```
yarn dev
# or
npm run dev
```

### Compiles and minifies for production
```
yarn build
# or
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).


## use docker

在项目根目录运行命令即可启动

```bash
docker-compose up -d
```

### 停止

```bash
docker-compose stop
```

### 删除所有未使用的卷

```bash
docker volume prune
```
